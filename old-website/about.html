เกี่ยวกับเรา

เรียนรู้เกี่ยวกับพันธกิจ ทีมงาน และสิ่งที่ขับเคลื่อนโปรเจค Workshop นี้

พันธกิจของเรา

เราสร้างทรัพยากรการเรียนรู้ที่เข้าถึงได้ง่ายและเป็นประโยชน์ เพื่อช่วยให้นักพัฒนาทำงานได้เร็วขึ้นและมีประสิทธิภาพมากขึ้น

โปรเจคนี้เริ่มต้นจาก Workshop แบบ hands-on เพื่อสอนเกี่ยวกับ Git, GitHub Actions และแนวทาง DevOps พื้นฐานที่ทุกคนสามารถนำไปปรับใช้ได้จริง

สิ่งที่เราทำ

  • ตัวอย่างที่ชัดเจน: มีตัวอย่างและแบบฝึกหัดที่เข้าใจง่ายสำหรับ version control
  • Template ที่ใช้งานได้จริง: Web templates ที่สามารถนำไปใช้ซ้ำได้ สร้างด้วย Bootstrap
  • CI/CD Workflows: สาธิตการทำงานของ CI/CD ด้วย GitHub Actions อย่างเข้าใจง่าย

ความชัดเจน

เราเชื่อในการสื่อสารที่ชัดเจน ทั้งในโค้ด เอกสาร และการสอน เพื่อให้ทุกคนเข้าใจได้ง่าย

การทำซ้ำได้

ทุกขั้นตอนสามารถทำซ้ำได้ และมีผลลัพธ์ที่คาดเดาได้ เหมาะสำหรับการนำไปปรับใช้จริง

เรียนรู้จากการทำ

เราเน้นการปฏิบัติจริง ด้วยการทำ testable changes เล็กๆ และเอกสารที่ช่วยให้ได้ผลลัพธ์เร็ว
